작품 핵심 분석 — 9가지 항목

아래 항목들을 하나씩 채우며 작품을 분석합니다. ①②③④는 필수 항목입니다.

①

화자의 정서 / 태도 필수

화자가 느끼는 정서와 태도를 서술합니다.
예) 「임에 대한 그리움과 체념, 연군지정」

②

주제 필수

작품의 핵심 주제를 한 문장으로 씁니다.
예) 「자연 속 은일과 임금에 대한 충성」

③

갈래 / 형식 분석 필수

갈래(시조·가사·향가 등) + 율격(3·4조/4·4조) + 연 구조를 정리합니다.

④

표현상 특징 필수

반복·대조·역설·설의·의인화·객관적상관물·감정이입·선경후정·시각적심상 등

⑤

핵심 시어 / 소재의 상징

중심 소재가 상징하는 의미를 분석합니다.
예) 「'대' → 절개와 지조 / '달' → 임금 또는 이상세계」

⑥

화자의 처지 / 배경 상황

화자가 처한 상황을 씁니다.
예) 「유배 중 / 낙향 후 강호생활 / 임과 이별」

⑦

임(대상)과의 관계

임=임금(연군지정) / 임=연인 / 자연=벗 / 자연=안식처 등

⑧

한자어 / 고어 핵심 어휘

어려운 고어·한자어를 정리합니다.
예) 「여희다 → 이별하다 / 세한고절 → 추운 계절의 절개」

⑨

배경지식 / 고사·전고

작품 이해에 필요한 배경지식을 정리합니다.
예) 「도연명 귀거래사, 상산사호, 굴원」

자주 묻는 질문 Student FAQ
Q. 앱을 닫으면 내용이 사라지나요?
아니요. 입력 내용은 브라우저에 자동 저장됩니다. 단, 브라우저 캐시를 지우거나 다른 기기에서 열면 사라집니다. 다른 기기에서 이어 쓰려면 💾 임시저장으로 JSON 파일을 받아 옮기세요.
Q. 셀프해설지 탭이 계속 잠겨 있어요.
모든 지문에서 ① 화자의 정서 또는 ② 주제 중 하나, 그리고 ③ 갈래/형식 또는 ④ 표현상 특징 중 하나를 반드시 입력해야 완료 처리됩니다. 지문 선택 드롭다운에서 아직 녹색 점(●)이 없는 지문을 찾아 채우세요.
Q. PDF 저장이 안 돼요.
일부 모바일 환경에서 PDF 생성이 실패할 수 있습니다. 이 경우 브라우저 인쇄 기능(Ctrl+P / ⌘+P) → PDF로 저장을 이용하세요.
Q. 제출 파일이 어디에 저장되나요?
📁 제출 저장 버튼을 누르면 다운로드 폴더에 「제출_이름_과제명.html」 파일이 저장됩니다. 이 파일을 카카오톡에서 파일 첨부로 전송하면 됩니다.
Q. 해설지 링크가 열리지 않아요.
해설지 링크는 모든 문제 도전을 완료한 후에만 공개됩니다. 아직 도전하지 않은 문항이 있으면 셀프해설지 탭에서 모두 완료하세요.
Q. 파일을 잃어버렸어요.
선생님께 카카오톡으로 파일을 다시 받으세요. 단, 새 파일을 받으면 이전에 작성한 내용은 사라지므로 처음부터 다시 작성해야 합니다. 작업 중 💾 임시저장 버튼을 자주 누르는 습관을 들이세요.
