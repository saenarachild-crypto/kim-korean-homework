📘 김선생님 디지털 작업실 (V13.1)

👇 작업 도구 바로가기 👇

🏭 일일 테스트 생성기 실행 📅 주간 과제 생성기 실행

Step 1. 준비물 챙기기

  • 작업 폴더: 바탕화면의 kim-korean-homework 폴더
  • 필수 프로그램: GitHub Desktop (배포 트럭)
  • 파일 준비: 테스트용 PDF나 지문 자료가 있다면 미리 폴더에 넣어두세요.

Step 2. 파일 만들기 (루틴)

1. 생성기 접속: 위 버튼 중 필요한 생성기를 누릅니다.

2. 정보 입력 (V13.1 핵심):

  • 제목: 정확하게 입력 (엑셀 시트 제목이 됩니다!)
  • 학원명: 이제 입력할 필요 없음 (학생이 입력함)
  • 문제/정답: 12345 처럼 정답만 입력

📢 중요 변경점 (V13.1):
더 이상 생성기에서 '학원'을 선택하지 않아도 됩니다.
만들어진 파일을 열면 학생이 직접 학원 이름을 입력하게 되어 있습니다.

3. 저장: [코드 생성] → [파일로 저장] 버튼 클릭!

4. 이동: 다운로드된 파일(예: test_0114.html)을 작업 폴더로 옮기세요.

Step 3. 배포하기 (GitHub Desktop)

1. GitHub Desktop을 켭니다.

2. 왼쪽 목록에 방금 넣은 파일들이 + 초록색으로 떴는지 확인합니다.

3. 왼쪽 아래 Summary 칸에 간단히 메모(예: 1월 14일 테스트)를 씁니다.

4. 파란색 [Commit to main] 버튼 클릭!

5. 오른쪽 위 [Push origin] 버튼 클릭! (로딩 바가 사라지면 배포 완료)

Step 4. 링크 보내기 & 확인

학생들에게 줄 주소 규칙은 변함없습니다.

https://kim-korean.netlify.app/파일명.html

[선생님이 할 일]

  • 엑셀(구글 시트)을 켜고, 새로운 과제 제목을 '1행'에 추가합니다.
  • 학생들에게 공지합니다: "학원명 띄어쓰기 틀리지 말고 정확히 적어라!"

💡 자주 묻는 질문 (FAQ)

Q. 점수가 엑셀에 안 들어와요.
A. 생성기 코드(HTML) 안에 선생님의 '구글 스크립트 주소'가 들어있는지 확인하세요.
또는 엑셀의 과제 제목(1행)과 생성할 때 적은 제목이 토씨 하나라도 다르면 안 들어옵니다.

Q. 학원 이름이 자꾸 늘어나요.
A. 걱정 마세요! V13.1 버전부터는 학생이 '시대인재', '강남대성' 등 무엇을 입력하든 엑셀에 그대로 찍힙니다.

Q. 수정했는데 학생들은 그대로래요.
A. 인터넷 브라우저 캐시 때문입니다. 학생들에게 "새로고침(F5) 하거나 창을 껐다 켜라"고 하세요.
