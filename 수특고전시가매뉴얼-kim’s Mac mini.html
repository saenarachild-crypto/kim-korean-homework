전체 개요 Overview
이 앱이란?

수능특강 고전시가 문제편을 공부하는 학생이 스스로 작품을 분석하고, 수능 문제에 도전하며, 분석 내용을 선생님께 제출하는 통합 학습 도구입니다. 선생님은 과제를 만들고, 학생은 분석 후 카카오톡으로 제출하면 됩니다.

👩‍🏫
선생님 모드
과제 생성기로 작품·문항·정답을 설정하고, HTML 파일을 만들어 카카오톡으로 배포합니다.
📖
학생 모드
받은 파일을 열어 작품을 분석하고, 도전 문제를 풀고, 분석 결과를 제출합니다.

과제 만들기 순서
1

🤖 AI 자동 추출 (선택)

수능특강 해설지 PDF를 업로드하면 Claude AI가 작품 정보·문항 번호·정답을 자동으로 분석합니다. 10~30초 소요됩니다. 자동 추출 후 내용을 확인하고 수정하세요.

2

📋 주간 과제 정보 입력

과제 제목, 마감 기한, 학습 기간, 선생님 메시지를 입력합니다.
예) 「2027 수능특강 고전시가 3월 1주차」

3

📜 작품 / 지문 목록 추가

각 지문마다 갈래, 작가, 제목, 페이지, 작품 수, 문항 수를 입력합니다.
복합 지문(가·나·다)은 작품 수를 조정하세요.

4

📝 수능 문제 설정 (선택)

문항 번호, 정답(①~⑤), 도전 횟수를 설정합니다. 설정하면 도전 시스템이 활성화됩니다.

5

🔗 링크 설정 (선택)

시험지 URL, 해설지 URL, Google Sheets 연동 URL을 입력합니다.
해설지 URL은 학생이 모든 도전을 완료한 후에만 공개됩니다.

6

📁 파일 생성 & 배포

「통합 파일 생성 & 저장」 버튼을 눌러 HTML 파일을 다운로드한 뒤, 카카오톡으로 학생들에게 전송합니다.

2. 작품 핵심 분석 (고전시가) — 9가지 항목
①

화자의 정서 / 태도

화자가 느끼는 정서와 태도를 서술합니다.
예) 「임에 대한 그리움과 체념, 연군지정」

②

주제

작품의 핵심 주제를 한 문장으로. 예) 「자연 속 은일과 임금에 대한 충성」

③

갈래 / 형식 분석

갈래(시조·가사·향가 등) + 율격(3·4조/4·4조) + 연 구조를 정리

④

표현상 특징

반복·대조·역설·설의·의인화·객관적상관물·감정이입·선경후정·시각적심상 등

⑤

핵심 시어 / 소재의 상징

중심 소재가 상징하는 의미 분석.
예) 「'대' → 절개와 지조 / '달' → 임금 또는 이상세계」

⑥

화자의 처지 / 배경 상황

화자가 처한 상황.
예) 「유배 중 / 낙향 후 강호생활 / 임과 이별」

⑦

임(대상)과의 관계

임=임금(연군지정) / 임=연인 / 자연=벗 / 자연=안식처 등

⑧

한자어 / 고어 핵심 어휘

어려운 고어·한자어 정리.
예) 「여희다 → 이별하다 / 세한고절 → 추운 계절의 절개」

⑨

배경지식 / 고사·전고

작품 이해에 필요한 배경지식.
예) 「도연명 귀거래사, 상산사호, 굴원」

하단 공통 입력 항목
📌

시어의 함축적 의미 + 추가 가능

핵심 시어 → 함축 의미를 키-값 형태로 여러 개 추가 가능

📚

어휘 정리 + 추가 가능

어휘 → 뜻풀이를 키-값 형태로 여러 개 추가 가능

💭

각 문제에 대해 정답 여부(○/△/✕)와 오답 이유를 기록

1

3

결과 확인

정답이면 🎉, 오답이면 남은 도전 횟수가 표시됩니다. 모든 도전 소진 시 정답이 공개됩니다.

도전 횟수 & 점수

각 문제마다 최대 도전 횟수(기본 3회)가 있습니다. 결과는 카카오 보고서에 반영됩니다.

상황결과점수 반영
1회에 정답 ✓ 정답 맞힘 정답으로 집계
2~3회 만에 정답 ✓ 정답 맞힘 정답으로 집계
도전 횟수 소진 ✕ 도전 종료 (정답 공개) 오답으로 집계
🔓
모든 도전이 완료되면 선생님이 설정한 해설지 URL 링크가 공개됩니다.

❓ 자주 묻는 질문
Q. 앱을 닫으면 내용이 사라지나요?
아니요. 입력 내용은 브라우저에 자동 저장됩니다. 단, 브라우저 캐시를 지우거나 다른 기기에서 열면 사라집니다. 다른 기기에서 이어 쓰려면 💾 임시저장으로 JSON 파일을 받아 옮기세요.
Q. 셀프해설지 탭이 계속 잠겨 있어요.
모든 지문에서 ① 화자의 정서 또는 ② 주제 중 하나, 그리고 ③ 갈래/형식 또는 ④ 표현상 특징 중 하나를 반드시 입력해야 완료 처리가 됩니다.
Q. PDF 저장이 안 돼요.
일부 모바일 환경에서 PDF 생성이 실패할 수 있습니다. 이 경우 브라우저 인쇄 기능(Ctrl+P / ⌘+P) → PDF로 저장을 이용하세요.
Q. 선생님 파일(생성기)과 학생 파일이 같은 파일인가요?
네. 원본 HTML 파일을 선생님이 생성기로 사용하고, 생성된 파일(과제 정보가 포함된 파일)을 학생에게 배포합니다. 제출된 파일(제출_이름_과제명.html)을 열면 자동으로 검토 모드가 됩니다.
Q. Google Sheets 연동은 어떻게 하나요?
이미 기본 Google Apps Script URL이 설정되어 있습니다. 학생이 제출 버튼을 누를 때마다 스프레드시트에 자동 기록됩니다. 별도 설정 없이 바로 사용 가능합니다.
